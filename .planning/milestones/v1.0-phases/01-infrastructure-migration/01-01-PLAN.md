<objective>
Migrate the build system and Cloudflare adapter from the archived @cloudflare/next-on-pages
(static Pages export) to @opennextjs/cloudflare (dynamic Workers runtime). This establishes
the adapter and config files that all subsequent plans in this phase depend on.

Purpose: Remove output: 'export' so dynamic API routes and server components run on Cloudflare
Workers. Install the OpenNext adapter, rewrite wrangler.toml for Workers, update package.json
scripts, fix the existing /api/leads route to use Workers env vars, and rewrite the deployment
tests to assert the new configuration.

Output: open-next.config.ts, rewritten wrangler.toml, updated next.config.mjs,
updated package.json, updated src/app/api/leads/route.ts,
rewritten src/tests/cloudflare-deployment.test.ts.

INFRA-01: @opennextjs/cloudflare replaces archived @cloudflare/next-on-pages.
INFRA-02: output: 'export' is removed; dynamic API routes work at runtime.
</objective>
