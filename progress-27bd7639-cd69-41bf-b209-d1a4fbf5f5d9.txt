**Learnings:**
- Dynamic routes in Next.js App Router use bracket syntax: [slug]
- generateStaticParams is REQUIRED for static export with dynamic routes
- generateStaticParams must return array of objects matching route parameters: { slug: string }[]
- notFound() from 'next/navigation' triggers the 404 page for invalid slugs
- Async page components can use generateMetadata separately (not in page component)
- Sticky positioning on desktop requires: lg:sticky lg:top-6 with parent positioning context
- Two-column layouts work well for detail pages: 2/3 for content, 1/3 for sticky sidebar
- Mobile-first responsive design: hide desktop elements with lg:hidden, show with hidden lg:block
- Property stats look better with icons (consistent with ListingCard)
- Price per sqft is a useful metric for buyers
- Helper functions for formatting (formatPrice, formatNumber) reusable across components
- Video sections should be conditionally rendered to avoid empty placeholders
